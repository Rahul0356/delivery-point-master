## API Reference

#### Get all Foods

```http
  GET /api/get-all-foods
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**.  api methods |

#### Get food by Id

```http
  GET /api/food/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch |
